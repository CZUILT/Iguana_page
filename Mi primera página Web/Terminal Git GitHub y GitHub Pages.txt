Terminal Git GitHub y GitHub Pages
mkdir proyecto1				: crea "documento o directorio" proyecto1
touch index.html			: crea "un archivo" index.html
cp index.html /proyecto1/index.html	: crea el archivo index.html dentro de carpeta proyecto1
cp index.html /proyecto1/index2.html 	: crea un nuevo archivo fuera de la carpeta, pero envia a la carpeta
cp index.html index3.html               : crea un nuevo archivo dentro de la misma carpeta
